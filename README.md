# NudgeBoo 👻

> **Nudge** (轻推) + **Boo** (幽灵叫声) = 一个会轻推你记录日志的小幽灵

## 🤡 灵魂拷问

- "今天干了啥？" "emmm...改了个bug？还是加了个功能？"
- 领导问你这周做了什么，你大脑一片空白
- 每天下班前都在想："我今天到底在干嘛？"

**恭喜你！NudgeBoo 就是来拯救你的！**

它是一个住在你桌面上的小幽灵，会时不时"Boo~"你一下，提醒你："喂！该写日志了！"

## ✨ 功能（说人话版）

### 👻 桌面小幽灵
- 会飘、会眨眼、会打瞌睡（和你一样）
- 戳它会弹一下（解压神器）
- 拖来拖去，想放哪放哪
- 状态随你的心情变化：开心、困成狗、惊恐...

### 📝 日志记录
- 双击幽灵 → 写日志
- 自动按日期保存（再也不用想"今天的文件叫啥"）
- 时间轴让你看看自己到底摸了多少鱼
- 支持子行编辑：双击已有记录可追加细节

### ✅ 待办事项
- 写待办 → 打勾 → 自动记录到日志
- 打勾那一刻，成就感拉满！
- 完成的任务会有删除线（看着就爽）

### ⏰ 催命提醒
- 定时戳你："该记日志了！"
- 间隔随你设（5分钟一催？你开心就好）
- 再也不能用"我忘了"当借口了

### 🔕 免打扰模式
- 开会了？一键免打扰，幽灵自动隐身
- 不弹通知、不显示幽灵，安安静静
- 随时取消，幽灵立刻回来

### ⚙️ 设置（藏得比较深）
- **工作模式**：固定上下班 or 弹性工作制（自己选）
- **午休时间**：摸鱼时间自己定
- **提醒间隔**：多久催一次（默认2小时）
- **日志路径**：建议存在领导找不到的地方

## 📁 和 Obsidian 完美配合

NudgeBoo 生成的 Markdown 文件可以直接在 Obsidian 里查看和编辑！

**推荐设置：**
1. 打开 NudgeBoo 设置（右键 → 设置）
2. 把日志路径改成你的 Obsidian 笔记库，比如：
   ```
   D:\MyVault\工作日志
   ```
3. 每天的工作记录会自动保存为 `MM.DD.md`
4. 在 Obsidian 里可以继续编辑、添加标签、链接笔记

**文件格式：**
```
工作日志/
├── 05.29.md
├── 05.30.md
├── 05.31.md
└── ...
```

这样你的工作日志就和 Obsidian 笔记库完美融合了！

## 📁 日志长啥样

每天一个文件，比如 `05.29.md`：

```markdown
# 2026/05/29 工作记录

10:30 改了个bug（又一个）
14:00 新增了功能（用户说不要了）
16:30 ✅ 完成待办: 写周报（编不下去了）
	细化了需求文档 [16:45]
	和产品经理对齐了 [17:00]
```

> 💡 双击已有记录可以追加子行细节，方便记录任务的各个步骤。

## 📂 数据文件说明

程序运行时会在 `data/` 目录下生成以下文件（不用管，自动管理的）：

```
data/
├── config.json      # 用户配置（工作模式、午休、提醒间隔等）
├── timeline.json    # 当日记录状态（自动维护，每天重置）
└── todos.json       # 待办事项列表
```

| 文件 | 说明 | 要不要手动改 |
|------|------|-------------|
| `config.json` | 所有设置项，修改后即时生效 | 可以，但建议走设置界面 |
| `timeline.json` | 记录状态、今日条目、上次记录时间 | 别动，它自己管自己 |
| `todos.json` | 待办事项纯文本数组 | 可以，改完重启生效 |

日志文件保存在设置的「日志路径」下（默认 `logs/`），每天一个 `.md` 文件。
